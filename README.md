# Duke-Shop
**Duke Shop Universal** to super-app łączący handel, gaming i social. Oferuje spiżarnię AR, zakupy w UE przez Passport i subskrypcje Steam. Całość wspiera AI Negotiator oraz system gildii z misjami. Kompletny ekosystem handlowy przyszłości, zaprojektowany w Gdańsku. To Twoje cyfrowe imperium w jednym kodzie.
